This pipeline runs the same checks as the PR pipeline with some important additions. This pipeline runs integration tests to test the project as a whole. If any of the checks fail, the code needs to be fixed before it can be prepared for deployment or delivery Continuous Deployment(CD) CI pipeline is also responsible for creating and building artifacts.
